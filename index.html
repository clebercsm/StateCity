Um Módulo JavaScript para gerenciamento de Estados e Cidades brasileiros.

Passo 1: Inserir dois elementos < select >

Passo 2: Importar Módulo StateCity < script type="module" >

Pronto, os elementos serão controlados pelo Módulo StateCity, incluindo validação caso uso o Bootstrap.

Dois arquivos são necessários, sendo eles:

  • Módulo JavaScript { sc-json.mjs }

  • Arquivo Json contendo Estados e Cidades brasileiros { statecity.json }
